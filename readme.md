# buddy

:dog: :cat: :mouse: :hamster:

Somos uma plataforma de gestão de serviços veterinários e passeio.

### instalação

```bash
git clone https://github.com/otallyto/buddy.git

cd buddy

npm install

npm start
```

### build 

```bash
docker build -t buddy .

docker run --name buddy -p 8000:3001 -d buddy
```
